Automated technical SEO auditing and fixing for WordPress sites with AI intelligence.

== Description ==

SEO Autopilot Pro is a revolutionary WordPress plugin that goes beyond traditional SEO tools like RankMath and Yoast. While those plugins simply check your SEO, SEO Autopilot Pro intelligently fixes issues using AI that understands your niche, audience, and brand voice.

= Key Features =

* **AI-Powered Site Profiling**: Automatically analyzes your site to understand your niche, target audience, writing style, and content strategy
* **Context-Aware Content Generation**: Creates meta descriptions, alt text, and content that matches your brand voice
* **Intelligent Auto-Fixing**: Automatically fixes SEO issues with PM/QA agent oversight for quality control
* **Comprehensive Auditing**: Integrates with Google Search Console, PageSpeed Insights, and more
* **Competitor Intelligence**: Analyzes competitor content to provide strategic recommendations
* **Automated Monitoring**: Continuous monitoring with alerts for critical SEO issues

= Why SEO Autopilot Pro? =

**RankMath and Yoast just CHECK things - SEO Autopilot Pro FIXES them intelligently**

* They're rule-based → We're strategically aware
* They don't understand your niche → We profile your entire site
* They can't write → We write at agency level with research
* They're static → We learn and adapt to your brand

= Powered by Advanced AI =

* Claude (Anthropic) for strategic content generation
* Gemini for image analysis and bulk keyword research
* PM/QA Agent system for quality control
* Web research integration for content creation

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/seo-autopilot-pro/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to SEO Autopilot → Settings to configure API keys
4. Run your first site profile to let AI understand your site
5. Start your first audit!

== Frequently Asked Questions ==

= Do I need API keys? =

Yes, you'll need API keys for:
* Claude API (Anthropic) - for content generation
* Gemini API (Google) - for image analysis
* Google Search Console - for indexation data
* PageSpeed Insights API - for performance metrics

= Will it work with my theme? =

Yes! SEO Autopilot Pro works with any WordPress theme and integrates directly with WordPress core functions.

= Can I review fixes before they're applied? =

Absolutely! You can enable manual approval mode, or let the AI PM/QA agent automatically review and approve high-quality fixes.

= How is this different from RankMath or Yoast? =

RankMath and Yoast are excellent for checking SEO issues, but they require you to fix everything manually. SEO Autopilot Pro:
* Automatically fixes issues
* Understands your site's niche and brand voice
* Researches topics before writing content
* Has a built-in QA system for quality control
* Provides competitor analysis and strategic recommendations
